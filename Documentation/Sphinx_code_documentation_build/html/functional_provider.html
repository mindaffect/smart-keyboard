functional_provider.py¶

This module defines the functionalities a key can be equipped with.

class functional_provider.FunctionalProvider¶

Bases: object

Interface to create a key functionality.

apply(key, window)¶

To be overridden by classes inheriting this interface.

Is called in handle_mouse_events of Key and applies this functionality when the equipped key is selected.

update(key)¶

To be overridden by classes inheriting this interface.

Is called in update of Key and updates the equipped key with changes in settings.

retrieve(key)¶

To be overridden by classes inheriting this interface.

Is called upon creation of a Key and provides the FunctionalProvider with necessary information.
