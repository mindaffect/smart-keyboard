dictionary_retrieval.py¶

This module is used to download word frequency dictionaries used for word correction. It contains the following functions:

dictionary_retrieval.check_dictionary(language)¶

Function to make sure the dictionary for the specified language is downloaded.

dictionary_retrieval.download(language, full)¶

Function to download dictionaries.
