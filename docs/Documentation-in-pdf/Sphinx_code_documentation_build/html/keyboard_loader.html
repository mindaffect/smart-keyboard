keyboard_loader.py¶

This module is for loading in keyboards / keypads layouts from JSON files.

It contains a single class KeyboardLoader.

A KeyboardLoader has the following functionality:

  • Static load_keyboard function which returns a list of objects that have been read from a JSON file. It contains all buttons the user wants to have on their keyboard.
