Installation & Usage¶

To use the keyboard, run the bci_keyboard.py file.

Requirements¶

This keyboard implementation is based on the mindaffectbci module developed by MindAffect so it should be installed.

pip install mindaffectBCI

The required packages can be found in requirements.txt and installed as follows.

pip install -r requirements.txt

We recommend using Python version 3.8

Problems¶

First check if your Python version is not outdated and if all the required packages have been installed correctly (see Requirements).
