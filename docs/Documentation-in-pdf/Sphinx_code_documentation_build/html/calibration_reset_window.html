calibration_reset_window.py¶

This modules contains a single class that shows text that calibration reset has been successful.

class windows.calibration_reset_window.CalibrationResetWindow(parent, facade, style, use_flickering, noisetag=None)¶

Bases: keyboard.windows.window.Window

Subclass of Window, designed for giving the user feedback that resetting the Calibration Model has been successful.

Parameters

  • parent (windows.window.Window) – The parent of this window.

  • facade (framework_facade.FrameworkFacade) – Contains the GUI specific functionality.

  • style (dict) – Style instructions for the keyboard.

  • use_flickering (bool) – Activates or deactivates flickering.

  • noisetag (noisetag.Noisetag) – Reference to the Noisetag module from MindAffect.

self.text¶

Framework dependent text object which holds the text.

Type

Object

activate()¶

Activates all visual and functional elements of the Window.

deactivate()¶

Deactivates all visual and functional elements of the Window.

draw(noisetag, last_flip_time, target_idx=- 1)¶

Changes the state of the buttons/keys within the Window. It draws the display with the colors given by the Noisetag module if flickering is active.

Parameters

  • noisetag (noisetag.Noisetag) – Reference to the Noisetag module from MindAffect.

  • last_flip_time (int) – Timestamp of last screen update, i.e. buffer flip.

  • target_idx (int) – (Optional) index of the target stimulus.
