psychopy_facade.py¶

This module is used as a facade between the bci keyboard and the psychopy framework.

This module contains a single class PsychopyFacade.

convert_to_icon_size(size, scale=1.0)¶

Converts given size to the size used for creating the icon.

The converted size is expressed in the coordinate system (-1,1) but ensures that the icon has the same height as width in pixels. The side length of this square is determined by the size of the smaller side of the key.

Parameters

  • size ((float, float)) – A size passed as a tuple of two fractions

  • scale (float, optional) – Scale of the icon

Returns

(float): The icon size
